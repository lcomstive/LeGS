LeGS
Tutorial 0 - Getting Familiar

LeGS is built on interfaces & events, with some simple implementations to get you started.

Interfaces

  • IEntitys represent LeGS objects in the game world.
  • IStatusEffects can be added and removed from an IStatusEffectReceiver
  • IStatusEffectReceiver is an IEntity that can have IStatusEffects added and removed
  • IDamageable objects have health and can receive damage.
  • ICharacter inherits from IEntity, IDamageable & IStatusEffectReceiver.

Events

The EventManager handles LeGS events, all of which will be derived from LEGEventArgs.

Events can be registered, subscribed and published.

Name Description
EntityHealthChangeEventArgs Damageable's health change
EntityDeathEventArgs Damageable's health has reached zero
StatusEffectChangeArgs A status effect has been added or removed from a receiver
AttributeEventArgs An IAttributeHolder's attribute has been added, removed or modified
QuestEventArgs Event for changes in a quest
